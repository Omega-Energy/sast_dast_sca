# Internal Binary Policy Service

AssemblyLine-сервис для проверки бинарных файлов на соответствие внутренним политикам.

## Проверки

- Наличие и валидность цифровой подписи
- Соответствие allowed-list вендоров
- Отсутствие запрещённых библиотек/зависимостей
- Проверка на известные уязвимые версии компонентов
