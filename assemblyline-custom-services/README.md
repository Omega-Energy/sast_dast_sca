# AssemblyLine Custom Services

Кастомные сервисы для платформы AssemblyLine (CCCS).

## Сервисы

| Сервис | Назначение |
|--------|-----------|
| `company-yara/` | YARA-сервис с корпоративными правилами |
| `vendor-reputation/` | Оценка репутации вендоров по базе |
| `internal-binary-policy/` | Проверка бинарей на соответствие внутренним политикам |

## Интеграция

Сервисы деплоятся как Docker-контейнеры в инстанс AssemblyLine
и автоматически обрабатывают сэмплы в пайплайне.
