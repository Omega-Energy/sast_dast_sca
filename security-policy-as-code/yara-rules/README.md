# YARA Rules

Правила для детекции вредоносного кода и подозрительных паттернов в бинарных файлах.

## Категории

- **malware/** — сигнатуры известного ВПО
- **suspicious/** — подозрительные паттерны (обфускация, упаковщики)
- **crypto/** — криптомайнеры, нестандартное использование crypto
- **supply-chain/** — индикаторы supply-chain атак
