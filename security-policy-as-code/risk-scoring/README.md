# Risk Scoring

Модель скоринга и приоритизации рисков.

## Факторы оценки

| Фактор | Вес | Описание |
|--------|-----|----------|
| CVSS Score | 0.3 | Базовый скор уязвимости |
| Exploitability | 0.25 | Наличие публичных эксплойтов |
| Asset criticality | 0.2 | Критичность затронутого актива |
| Exposure | 0.15 | Доступность из интернета |
| Data sensitivity | 0.1 | Чувствительность обрабатываемых данных |

## Выходные уровни

- **CRITICAL** (9.0–10.0) — немедленное реагирование
- **HIGH** (7.0–8.9) — исправить в течение 24ч
- **MEDIUM** (4.0–6.9) — исправить в текущем спринте
- **LOW** (0.1–3.9) — бэклог
