# Portal UI

Web-интерфейс платформы безопасности.

## Стек

- React + Vite
- TailwindCSS
- Lucide Icons
- WebSocket для real-time логов

## Функционал

- Дашборд со сводной статистикой
- Запуск и мониторинг сканов
- Просмотр результатов с фильтрацией
- Управление проектами и коннекторами
- Сравнение сканов
