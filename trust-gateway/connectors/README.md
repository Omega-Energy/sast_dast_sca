# Connectors

Модули интеграции с внешними системами безопасности и CI/CD.

## Поддерживаемые интеграции

| Система | Тип | Статус |
|---------|-----|--------|
| GitLab CI | Webhook + API | Планируется |
| SonarQube | API (импорт результатов) | Планируется |
| Cuckoo Sandbox | API (отправка сэмплов) | Планируется |
| AssemblyLine | API | Планируется |

## Архитектура

Каждый коннектор реализует единый интерфейс `BaseConnector`:
- `connect()` — установка соединения
- `sync()` — синхронизация данных
- `webhook_handler()` — обработка входящих событий
